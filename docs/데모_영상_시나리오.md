## 핵심 메시지 (심사 포인트)
- **작동 증명**: 입력 1회 → 카드 한 장(백분위·특기·로드맵·상담)이 실제로 나온다.
- **정직성**: 위험질문(개인판정·합격예측·의료진단)을 게이트가 거부하고 우회로를 준다.
- **공공데이터**: 병무청 7종 실사용(백분위·특기·경쟁률·복무기관·소집·공석·진로센터).
- **측정된 AI**: `eval/RESULTS.md`에 dev/held-out 수치가 파일로.
